HTML Cheat Sheet

<html>: starts the HTML document

<body>: starts the main body of the document

<head>: a space for various document configuration stuff

<p>: gives you a line for various typing

<h#>: gives you a heading

<b>: makes enclosed text bold

<i>: makes the enclosed text italicized

NOTE: to close tags, add a / before the tag name (eg: /html)

HTML Entities:

Entities are special codes made to represent characters not normally supported in HTML. An entity is an ampersand followed by a semicolon, with a code in between.

To put an image, such as the one above, you must use the tag <img src="filename.extension">

NOTE: you MUST have the image in the directory of the page, or a subfolder in that directory. otherwise, the image WILL NOT LOAD!!!!

Some attribute stuff:

An attribute is more than just a tag. Attributes go in tags to make them do stuff, or be stuff, such as a image or a hyperlink.
