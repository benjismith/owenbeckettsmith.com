html Cheat Sheet

<html>: starts the html document

<body>: starts the main body of the document

<head>: a space for various document configuration stuff

<p>: gives you a line for various typing

<h#>: gives you a heading

<b>: makes enclosed text bold

<i>: makes the enclosed text italicized

NOTE: to close tags, add a / before the tag name (eg: /html)

HTML Entities:

Entities are special codes made to represent charicters not normally supported in html. An entity is an ampersand followed by a semicolon, with a code in between.
